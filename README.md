## サービス概要

**第八終活リーガルクリニック**は、高齢者が終活に必要な手続きを確認するためのWebアプリケーションです。

病院の総合受付（問診票）をイメージした入力インターフェースで、ご自身の状況を入力するだけで以下の書類・情報を自動生成します。

- 📋 **相続人関係図**（法定相続分を分数で図示）
- 💰 **財産目録**（7カテゴリ、印刷対応）
- 📓 **エンディングノート**（医療希望・葬儀希望・重要書類の保管場所など）
- 📄 **生前契約一覧**（死後に解約が必要な契約27種類）
- 💊 **処方箋**（入力内容に応じた個別アドバイス）

---

## 機能一覧

### 問診票（5ステップ入力）

| ステップ | 内容 |
|---|---|
| 家族構成 | 婚姻状況・子の人数・両親の状況・兄弟姉妹の人数・代襲相続の確認 |
| 財産目録 | 不動産・預貯金・有価証券・保険・車両・その他財産・負債 |
| 対策状況 | 遺言書・任意後見・死後事務委任など実施済の対策を確認 |
| お悩み | 遺産・認知症・葬儀・身元保証など現在の不安を確認 |
| ノート | エンディングノート＋生前契約一覧の入力 |

### 診察結果（自動生成）

- **相続関係図**：直系尊属は本人の上・直系卑属は下・傍系血族は横に配置。棒線で接続、法定相続分を分数表記
- **処方箋**：入力内容に応じた優先度付きアドバイス
- **専門家紹介**：必要な専門家の種類（司法書士・弁護士・税理士等）を提示

### 印刷機能

「印刷プレビュー」ボタンから財産目録・エンディングノート・契約一覧を一括でプレビュー表示。`Ctrl+P`（Mac: `⌘+P`）で印刷できます。専門家へのご相談時の持参資料としてご利用ください。

## GitHub Pages への公開手順

1. このリポジトリをご自身のGitHubアカウントにフォーク or クローン
2. **Settings** → **Pages** を開く
3. **Source** で `Deploy from a branch` を選択
4. **Branch** で `main`（または `master`）・`/ (root)` を選択して **Save**
5. 数分後に `https://[username].github.io/[repository]` で公開されます

### カスタムドメインを使う場合

1. **Settings** → **Pages** → **Custom domain** にドメインを入力
2. ドメインのDNS設定でCNAMEレコードを `[username].github.io` に向ける
